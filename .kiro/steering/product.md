# CardDemo Product Overview

CardDemo is a comprehensive mainframe credit card management application designed to showcase AWS and partner technologies for mainframe migration and modernization scenarios. It simulates a realistic credit card processing system with customer, account, card, and transaction management capabilities.

## Purpose
- Demonstrate mainframe application patterns and technologies
- Provide a realistic environment for testing migration and modernization approaches
- Support application discovery, analysis, and transformation tooling
- Enable performance testing and system augmentation scenarios

## Core Features
- Customer and account management
- Credit card lifecycle management
- Transaction processing and reporting
- Bill payment functionality
- User security and role-based access (Admin/User)
- Statement generation

## Optional Extensions
- **Credit Card Authorizations**: Real-time authorization processing with IMS DB, DB2, and MQ integration
- **Transaction Type Management**: DB2-based transaction type maintenance
- **Account Extractions**: MQ-based account data extraction and transmission
- **Enhanced Data Formats**: Support for complex COBOL data structures and various dataset types

## User Types
- **Regular Users**: Standard card management functions
- **Admin Users**: User management and administrative functions

The application intentionally incorporates various coding styles and patterns to exercise analysis, transformation, and migration tooling across different mainframe programming paradigms.
